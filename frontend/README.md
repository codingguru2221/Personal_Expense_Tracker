# Personal Expense Tracker - Frontend

This is the frontend for the Personal Expense Tracker application built with React, TypeScript, and Vite.

## Features

- User authentication (signup/login)
- Dashboard with financial analytics
- Account management
- Transaction tracking
- Loan management
- Responsive UI with Tailwind CSS

## Prerequisites

- Node.js and npm

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Application**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Development

The frontend connects to the backend API at `http://localhost:8080`. Make sure the backend is running for full functionality.

For easy startup of both frontend and backend, see the [RUNNING.md](../RUNNING.md) file in the root directory.
